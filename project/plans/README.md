# Development Plans

## Active Plans
Current development phase plans being executed.

Location: `active/`

## Archived Plans
Completed or superseded plans for reference.

Location: `archive/`

## Templates
Standard templates for creating new plans.

Location: `templates/`

## Plan Structure

Each plan should include:
1. **Objective** - What we're building and why
2. **Scope** - What's included and excluded
3. **Tasks** - Breakdown of work items
4. **Timeline** - Expected schedule
5. **Dependencies** - What's needed
6. **Risks** - Potential issues and mitigations
7. **Success Criteria** - How we measure completion
