## Adding Items

When adding new backlog items, include:
- **Title**: Clear, concise name
- **Description**: What needs to be done
- **Priority**: P0 (critical) to P3 (nice-to-have)
- **Estimate**: Rough size (S/M/L/XL)
- **Dependencies**: What's needed first

## Size Legend
- **S** (Small): < 1 day
- **M** (Medium): 1-3 days
- **L** (Large): 1-2 weeks
- **XL** (Extra Large): > 2 weeks
